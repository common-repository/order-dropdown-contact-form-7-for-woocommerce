== Description ==

<Strong>Order Dropdown WooCommerce For Contact Form 7</Strong> Making Order List Field to you conact form. This field creates a drop down of current user's orders. Via this plugin You can be make your contact form 7 to more professional.

For multiple order status need to put using | (bar).
Example field: 
<code>
[woocommerce_order woocommerce_order-968 status:wc-processing|wc-on-hold]
</code>
WooCommerce status are: wc-pending, wc-processing, wc-on-hold, wc-completed, wc-cancelled, wc-refunded, wc-failed
don't use the attribute 'status'.
